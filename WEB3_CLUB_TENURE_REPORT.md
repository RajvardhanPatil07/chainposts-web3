# Web3 Club Tenure Report

## Role
Web3 Club Member

## Overview
During my tenure in the Web3 club, I worked on hands-on blockchain development with a focus on building and testing decentralized applications. My main project was **ChainPosts**, a simple decentralized social posting application where users can connect their wallet and publish short posts stored directly on-chain.

## Tasks Completed
- Built a smart contract in Solidity for storing short posts on-chain.
- Added post validation logic such as empty-post checks and maximum byte limits.
- Wrote contract tests to verify post creation and retrieval behavior.
- Integrated wallet connection using MetaMask.
- Built a frontend for connecting wallets, creating posts, and viewing recent on-chain posts.
- Configured local blockchain development using Hardhat.
- Fixed wallet/RPC issues during transaction submission and improved error handling.
- Added responsive UI improvements, dark/light mode, and branding updates.
- Deployed the frontend on Vercel for public access.

## Project Completed
**ChainPosts DApp**
- Tech stack: Solidity, Hardhat, React, Vite, Ethers.js, MetaMask, Vercel
- Core idea: A lightweight decentralized application for publishing short posts directly to a smart contract.
- Features delivered:
  - Wallet connection
  - On-chain post publishing
  - Live post feed from the contract
  - Local deployment and hosted frontend deployment

## Concepts Learned
- Smart contract structure, storage, events, and custom errors
- Writing and running blockchain tests with Hardhat
- Connecting frontend applications to Ethereum wallets
- Reading from and writing to contracts using Ethers.js
- Managing chain IDs, RPC URLs, contract addresses, and deployment flow
- Handling real-world Web3 issues such as wallet errors, insufficient balance, and network mismatch
- Importance of combining blockchain logic with usable frontend design

## Outcome
This tenure helped me strengthen both my **Web3 development fundamentals** and my **practical project-building skills**. I gained experience in taking a dApp from contract development to frontend integration, debugging, and deployment.
